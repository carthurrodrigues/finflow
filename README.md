# FinFlow - Auth Service

FinFlow is a distributed payment processing platform inspired by real-world fintech backend architecture.

This repository contains the Auth Service, responsible for authentication, authorization, RBAC modeling, and security-related user identity management.

## Current Stage

Work in progress.

Current focus:
- Java 21
- Spring Boot
- PostgreSQL
- Flyway migrations
- Docker Compose local infrastructure
- Clean Architecture package structure
- RBAC foundation with users, roles, and permissions

## Planned Architecture

FinFlow will evolve into a distributed platform composed of:

- API Gateway
- Auth Service
- Payment Service
- Fraud Detection Service
- Ledger Service
- Notification Service
- Audit Service
- Analytics Service

## Engineering Goals

- Clean Architecture
- Domain-Driven Design
- Event-driven architecture
- Kafka-based asynchronous communication
- Saga Pattern
- Outbox Pattern
- Idempotency
- Observability with OpenTelemetry, Prometheus, Grafana, and Loki
- Secure authentication and authorization
- PostgreSQL schema versioning with Flyway

## Status

Initial foundation in progress.
